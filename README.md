# Macroeconomics Tools for Business Administration

### Giới thiệu
Dự án này chứa các bài tập thực hành Python về Kinh tế vĩ mô trong chương trình học Quản trị doanh nghiệp tại Đại học Thương Mại (TMU).

### Các tính năng chính:
* Tính toán GDP theo phương pháp chi tiêu.
* Phân tích tỷ lệ lạm phát dựa trên chỉ số CPI.
* Hỗ trợ ra quyết định trong quản trị dựa trên dữ liệu vĩ mô.

### Cách chạy:
1. Cài đặt Python 3.x
2. Chạy lệnh: `python main.py`
